## ADDED Requirements
### Requirement: Website Footer
The system SHALL provide a global footer that appears at the bottom of the main website pages, containing brand info, location, contact methods, and legal links.

#### Scenario: User views brand and social info
- **WHEN** the user scrolls to the footer
- **THEN** they see the brand logo/name, a short description of the services, and social media links

#### Scenario: User needs to find the clinic location
- **WHEN** the user looks at the "Ubicación" column
- **THEN** they see the physical address and a visual map reference

#### Scenario: User desires to book an appointment
- **WHEN** the user checks the "Contacto Directo" column
- **THEN** they see working hours and a prominent WhatsApp reservation button

#### Scenario: User looks for legal information
- **WHEN** the user looks at the very bottom of the page
- **THEN** they see copyright text, privacy policy, and legal notice links
