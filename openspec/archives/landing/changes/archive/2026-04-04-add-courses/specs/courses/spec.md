## ADDED Requirements
### Requirement: Courses Display
The system SHALL provide a showcase section for Academy Courses, displaying the class level, duration, title, description, and a representative optimized image.

#### Scenario: Displaying multiple courses
- **WHEN** the user visits the homepage
- **THEN** they are presented with a grid of available courses

#### Scenario: Course details
- **WHEN** observing a single course card
- **THEN** an explicit badge highlighting the difficulty level ("Inicial", "Avanzado") is visible alongside standard details
