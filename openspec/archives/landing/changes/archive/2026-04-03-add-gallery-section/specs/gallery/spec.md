## ADDED Requirements
### Requirement: Gallery Section
The system SHALL provide a gallery section to display images of the provided services.

#### Scenario: Display image grid
- **WHEN** a user visits the page containing the gallery
- **THEN** they see an organized, structured grid of high-quality service images.

#### Scenario: Responsive layout
- **WHEN** a user views the gallery on a mobile device
- **THEN** the grid adapts its column count to ensure images remain visible and well-proportioned.
