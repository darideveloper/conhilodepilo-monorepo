# Proposal: Add Global Custom Scrollbar

## Why
The project currently uses the browser's default scrollbar, which does not match the custom branding and design system, leading to a less polished UI experience.

## What Changes
- `src/styles/global.css`: Add global scrollbar base styles using project brand variables.

## Objectives
- Apply a custom scrollbar style globally.
- Use project brand colors (`--color-brand-primary`, `--color-brand-secondary`) and UI spacing/radius variables.
- Ensure the scrollbar is accessible and visually consistent.

## Scope
- `src/styles/global.css`: Add global scrollbar base styles.
